# Politica de Seguranca

Se voce identificar uma vulnerabilidade de seguranca no `labiia_lex`, nao abra um
issue publico de imediato.

Envie o relato para:

- `cardososampaio@gmail.com`

Inclua, se possivel:

- descricao objetiva do problema
- impacto esperado
- passos de reproducao
- versao do aplicativo
- evidencias ou arquivos minimos necessarios

Observacoes:

- este projeto e mantido em capacidade academica e pode nao ter SLA formal
- faremos o possivel para confirmar o recebimento e avaliar a gravidade
